# Contributing to tokyo-nova-axleration

このプロジェクトへの関心をありがとうございます。

本プロジェクトは長期戦を前提とした個人主導の開発であり、現時点では基盤整備を進めている段階です。Pull Request やバグ報告は歓迎しますが、マージ判断は慎重に行います。

## 開発の進め方

本プロジェクトは段階的なロードマップに沿って開発しています。現在のフェーズとロードマップは [CLAUDE.md](./CLAUDE.md) を参照してください。

現在のフェーズに該当しない範囲の機能追加 PR は、設計の整合性の観点からマージできない場合があります。大きな変更を提案する前に、まず Issue で相談してください。

## バグ報告

バグを発見した場合は、GitHub Issues に以下の情報を添えて報告してください。

- Foundry VTT のバージョン
- 本システムのバージョン
- 再現手順
- 期待される動作と実際の動作
- 関連すると思われるコンソールエラー(あれば)

既知の不具合は [docs/KNOWN_ISSUES.md](./docs/KNOWN_ISSUES.md) にまとめられているので、報告前にご確認ください。

## Pull Request の方針

### 受け付ける PR

- 既知の不具合(`docs/KNOWN_ISSUES.md` 記載)の修正
- ドキュメントの誤字・脱字修正、表現改善
- 翻訳の追加・改善(`lang/` 配下)

### 慎重に検討する PR

- 既存機能の挙動を変更するもの
- データモデル(`template.json` / 将来の `DataModel`)に手を加えるもの
- 新規機能の追加

これらは、まず Issue で設計意図を相談してから着手することを強くおすすめします。一方的に大きな変更を含む PR を送られても、マージできない可能性があります。

### 受け付けられない PR

- **ゲームルールのテキストやデータの追加**(後述の著作権上の理由)
- システム全体のリファクタリング・書き換え
- 動作確認なしの大規模変更

## 著作権に関する重要な注意

本プロジェクトは「トーキョーN◎VA THE AXLERATION」の二次創作物です。元ゲームの著作権者(「鈴吹太郎」「有限会社ファー・イースト・アミューズメント・リサーチ」および「株式会社KADOKAWA」)の権利を尊重するため、本リポジトリには以下を含めない方針を取っています。

- ルールブックからの転載・引用(数値表・特技説明・神業効果文・シナリオ等を含む)
- スタイル名・特技名等の網羅的なリスト(構造的な enum として最小限必要なものを除く)
- 公式書籍掲載のイラスト・画像
- 公式書籍掲載のフレーバーテキスト

Pull Request にこれらを含めないでください。意図せず含まれていた場合、レビュー時に削除を依頼します。

判断に迷う場合は、PR を出す前に Issue で相談してください。

## コーディング規約

- ファイル名: kebab-case (`tnx-cast-sheet.mjs`)
- クラス名: PascalCase (`TokyoNovaCastSheet`)
- パッケージID・フラグスコープは `system.json` を踏襲

詳しくは [CLAUDE.md](./CLAUDE.md) の「開発ルール」セクションを参照してください。
